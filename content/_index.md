# Deploying a Java Spring Boot Application on EC2 within a VPC

### Overall
 This workshop is designed to provide participants with knowledge and practical experience in deploying a Java Spring Boot application on an EC2 instance within a Virtual Private Cloud (VPC). The session will focus on practices for secure and scalable deployments in a controlled network environment, leveraging AWS services.
